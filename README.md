# 📖 Overview

**ZOGO** is a modern full-stack grocery delivery platform inspired by Blinkit, Zepto and Instamart.

The application is specially designed for customers in **Midnapore District, West Bengal**, offering local grocery delivery with a premium shopping experience.

The platform includes customer ordering, authentication, shopping cart, checkout, order management and a professional admin dashboard.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/zogo.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

# 🌐 Environment Variables
